# Sorting Visualizer

An interactive browser-based Sorting Algorithm Visualizer built using HTML, CSS, and JavaScript.  
This project visually demonstrates how different sorting algorithms work through animated bar representations.

It is designed as a resume project to showcase front-end development skills and understanding of Data Structures & Algorithms.

---

## 🚀 Features

### 📊 Algorithms Implemented
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Counting Sort
- Radix Sort
- Bucket Sort

### 🎯 Functional Highlights
- Adjustable array size
- Adjustable animation speed
- Generate new random arrays
- Start / Pause / Resume / Reset controls
- Visual color cues for:
  - Comparisons
  - Swaps
  - Sorted elements

---

## 🖥️ Usage

1. Open `index.html` in any modern web browser.
2. Select a sorting algorithm from the dropdown.
3. Adjust the **Size** slider to change number of elements.
4. Adjust the **Speed** slider (lower value = faster animation).
5. Click **Generate Array** to create a new dataset.
6. Press **Start** to run the visualization.

> Tip: Lower speed value results in faster animations.

---

## ⚙️ Customization

- Modify `script.js` to add more algorithms or tweak animation logic.
- Update styles and colors inside `style.css` to customize UI appearance.
