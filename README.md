# Sorting Visualizer

This is a browser-based sorting algorithm visualizer created as a major project for a resume.
It demonstrates the behaviour of many common (and some advanced) sorting algorithms with animated bars.

## Features

- **Algorithms included**:
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort
  - Heap Sort
  - Shell Sort
  - Cocktail Sort
  - Comb Sort
  - Gnome Sort
  - Counting Sort
  - Radix Sort
  - Bucket Sort

- Adjustable array size and animation speed
- Generate random arrays with a single click
- Pause / resume / reset controls
- Visual cues for comparisons, swaps and sorted elements

## Usage

1. Open `index.html` in a modern browser (no server required).
2. Choose an algorithm from the dropdown.
3. Adjust the `Size` slider to change number of elements.
4. Adjust the `Speed` slider (lower = faster animation).
5. Click **Generate Array** to refresh.
6. Press **Start** to run the selected sort and watch the visualization.

> Tip: Lower speed value = faster animation (slider is inverted).

## Customization

You can modify `script.js` to add more algorithms or tweak colours in `style.css`.
